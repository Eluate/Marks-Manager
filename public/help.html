Registration

The registration menu contains two fields that require input in order to register. Registration creates an account that may be accessed using the respective username and password, enabling the usage of the manager.


The Username field requires input of 4-16 alphanumeric characters long. Alphanumeric characters consist of A-Z, a-z and 0-9.

The Password field requires input of 6-16 alphanumeric characters long. Characters typed in are shown as asterisks (*) to protect your password from onlookers. You should keep your password secure as it prevents your account from being accessed by others.

That dialogue box confirms that you have successfully completed the registration process.

That dialogue box means that somebody else has registered with that username. To correct this, you need to use a different username.



Signing In

Signing in allows your account to be accessed and modified. Signing in requires a username and password pair that was used in the registration process.


The Username requires the username inputted when registering.

The Password requires the password inputted when registering. Characters typed in are shown as asterisks (*) to protect your password from onlookers. You should keep your password secure as it prevents your account from being accessed by others.

That dialogue box means that you have entered an invalid username and password combination for an account.


Creating an Assessment

Creating an assessment allows it to be viewed in different ways. Collectively, created assessments can show a schedule of your assessment due dates and provide information on subject performance.


The Manage -> Create interface contains four fields that require input and one field that can have input optionally. The interface is accessed by clicking on Create under Manage.


The Task Name field requires input of 1-50 characters long. It should be the name of the task.

The Subject Name field requires input of 1-50 characters long. The entered value should be the name, preferably an acronym of the course that the assessment belongs to. To display all related information under the Subject View, the subject entered should be identical to the other assessments of the same course.

A calendar opens up when date due is pressed. Clicking on the arrows changes the month. Clicking on a day chooses that day as the date when an assessment is due.

The Weighting input should be the percentage of which the mark you receive impacts your final mark.

The Mark input is optional and should be the percentage mark, out of 100, that you receive for an assessment.



Deleting an Assessment

Removes an assessment from an account.


The Subject Name field is a dropdown that shows all of your subjects, generated from your assessment entries.

The Task Name field is a dropdown that shows all of the assessment task names that belong to the selected subject.



Viewing an Assessment

There are multiple ways that you can view data generated from your assessment input.


Selecting View -> Subjects displays a summary of your performance in your subjects. It is automatically shown after signing in.


Selecting View -> Tasks displays information about every assessment task you have created.


Selecting View -> Calendar displays a calendar and plots the due dates of all of your assessments on it.
